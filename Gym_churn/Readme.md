Проект, помогающий сети фитнес-центров разрабатывать стратегию взаимодействия с клиентами на основе аналитических данных и предлагающий варианты решения распространённой проблемы фитнес-центров - оттока клиентов.

Цель проекта: провести анализ предоставленных данных и, используя средства машинного обучения, спрогнозировать вероятность оттока клиентов на уровне следующего месяца и подготовить план действий по удержанию клиентов.
