### Изучение закономерностей, определяющих успешность компьютерных игр

__Задачи проекта:__ Используя исторические данные о продажах компьютерных игр, оценки пользователей и экспертов, разделение на жанры и платформы, выявить закономерности, определяющие успешность игры.

__Описание проекта:__ Выявлены параметры, определяющие успешность игры в разных регионах мира. На основании этого для магазина компьютерных игр подготовлен отчёт, предназначенный для более точного планирования рекламных кампаний. Проведена предобработка данных и их последующий анализ. Выбран актуальный период для анализа. Составлены общие портреты пользователей каждого региона. Проверены гипотезы: 
- средние пользовательские рейтинги платформ Xbox One и PC одинаковые;
- средние пользовательские рейтинги жанров Action и Sports разные.
