# Practicum_projects
Учебные проекты, созданные во время обучения на курсе Аналитик данных.

[__Games__](Practicum_projects/Games/Games_final.ipynb) - проект-исследование факторов, влияющих на успешность компьютеоных игр. Проверка гипотез. _Стек: Pandas, Matplotlib, Seaborn._

[__GoFast__](GoFast/_gofast_final_ipynb.ipynb) - анализ данных пользователей сервиса аренды самокатов "GoFast" для проверки гипотез, способствующих росту бизнеса. _Стек: Pandas, Matplotlib, Seaborn, Numpy._
